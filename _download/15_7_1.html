15.7 Number オブジェクト

15.7.1 関数として呼ぶ Number コンストラクタ

　 Numberが、コンストラクタとしてではなく、関数として呼ばれる際には、型変換を実行します。

15.7.1.1 Number ( [ value ] )

　 value が提供された場合、 ToNumber(value) によって算出した(Numberオブジェクトではなく)Number値を返し、それ以外の場合には、+0を返します。
